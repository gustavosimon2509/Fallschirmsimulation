#### Fallschirmsimulation
Dieses Projekt ist ein interaktives Werkzeug zur Simulation von Fallschirmflügen. Es ermöglicht die Analyse von Flugdaten durch eine grafische Benutzeroberfläche (GUI) und den Vergleich verschiedener Szenarien.


## Features
Datenvisualisierung: Erstellung präziser Flugbahnen und Grafiken.
- Echtzeit-Bearbeitung: Parameter können direkt in der User-fernster geändert werden.
- Vergleichsmodus: Mehrere Simulationen gleichzeitig analysieren.
- Dark Mode: Integrierter Dark Mode für eine angenehme Arbeitsumgebung.


## Erste Schritte
# Installation
1. Repository klonen:

	C:> git clone https://github.com/gustavosimon2509/Fallschirmsimulation.git

2. In den Projektordner wechseln:

	C:> cd fallschirmsimulation


3. Das Programm starten

	python src/UserInterface.py


## Bedienungsanleitung
Sobald die Benutzeroberfläche geöffnet ist, haben Sie folgende Möglichkeiten:

1. Simulation auswählen:
	Klicken Sie in der Hauptliste auf eine Simulation, um sie zu markieren.

2. Simulationen vergleichen:
	Möchten Sie mehrere Szenarien gleichzeitig analysieren? Klicken Sie auf die Schaltfläche "Simulationen vergleichen" und Wählen Sie mehrere Einträge aus.

3. Parameter einsehen:
	Klicken Sie auf den Button "Simulation Details", um alle technischen Daten der gewählten Simulation anzuzeigen.

4. Werte bearbeiten:
	Um einen Parameter (z.B. Gewicht oder Apogeum) zu ändern, machen Sie einfach einen Doppelklick auf den entsprechenden Eintrag in der Detail-Liste. Geben Sie den neuen Wert ein und bestätigen Sie.

5. Simulation starten:
	Wenn Sie bereit sind, klicken Sie auf den großen roten Button "Run". Die Berechnungen werden gestartet und die Grafiken generiert.
